# Georgia Travel

Travel booking platform for Georgia tours.

## Features
- Public website with tour listings
- Admin panel for tour management
- CSV/Excel data import
- Booking system

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Create `.env.local`:
```
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key
```
